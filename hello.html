Pass a natural date (January 1, 2016) or a timestamp (1450137600) as a parameter and returns json containing both. If no valid input, returns json containing null to both fields.

Examples :


https://%%URL%%/September%2009%202001
https://%%URL%%/September 09, 2001
https://%%URL%%/999993600

Returns :

{"unix":999993600,"natural":"September 09, 2001"}


https://%%URL%%/Hello
https://%%URL%%/

Returns :

{"unix":null,"natural":null}
